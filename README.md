## Setup instructions

1. Clone this repo with `git clone https://github.com/ed-donner/cma.git`
2. [Install uv](https://docs.astral.sh/uv/getting-started/installation/) if you don't already have it
3. From VS Code or Cursor, do File >> New Window and then Open.. then navigate into `cma` and Open
4. Open a Terminal window (View menu >> Terminal) and run `uv sync`
5. In the File Explorer on the left, head over to `demo.ipynb` then click on the top right to Select Kernel and pick `cma (Python 3.13.x)`
